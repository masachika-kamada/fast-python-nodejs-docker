# Fast Python-Nodejs Docker

## Overview

Python と Nodejs の Docker 開発環境。Big Mount によるホスト側とのファイル同期により Node の実行速度が急激に遅くなるため、`/workspace` 配下ではなく `/home` 配下にプロジェクトが作成されるようにしてある。

## Usage

VS Code の DevContainers でコンテナを起動する
